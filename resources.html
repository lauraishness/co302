HTML/CSS Resources

Templates & Themes

StartBootstrap Themes

These themes tend to have more styling and tend to include more images.

HTML5up Themes

These themes also have more style and tend to include more images; many of the themes here are free -- some are paid; you shouldn't have to purchase a theme for A2.

StartBootstrap Templates

These templates tend to be more "barebones" -- they have less styling than the themes above, but still incorporate the responsiveness of the bootstrap framework.

Snippets of Code

These elements work best with the start bootstrap themes/templates above, but they might also work generally for other templates that use bootstrap. These snippets have various elements that you might use.

General Guides/Tools

Codecademy Tutorials for HTML/CSS

These are the exercises that were assigned in Week 5; you can sign up for an account for free here and access tutorials about HTML/CSS and other developer tools.

W3Schools Tutorials

w3schools offers comprehensive and usable guides for basic HTML, CSS, and other front-end languages. It is run by the World Wide Web Consortium (founded by Tim Berners-Lee), which is THE main standards organization for web content and development; while there are other relevant regulatory and standards bodies, W3C is an authority on content standards.

CSS Basic Tutorial

This basic overview of CSS describes the difference between inline, internal, and external style sheets; the guide also gives a brief introduction to the syntax of CSS.

CSS Full Tutorial

This site, along with the pages linked on the sidebar, offer a more comprehensive overview of stylesheets.

CSS Properties List

This "dictionary" of CSS properties gives a general overview of the properties that you can style with CSS.

HEX Codes for Colors in HTML

This site allows you to snag the hex code for a particular color; by clicking on "more info", you can also snag the rgb code for CSS. Read more about colors and CSS on this W3schools tutorial.
